# 🔹 BLOQUE 2 – GROUP BY Y HAVING

## 1️⃣ Número de empleados por departamento

```sql
select id_departamento, count(*)
from empleados
group by id_departamento;
```

✔ Correcto

## 2️⃣ Salario medio por departamento

```sql
select id_departamento, avg(salario)
from empleados
group by id_departamento;
```

✔ Correcto (ajustada posición columnas)

---

## 3️⃣ Departamentos con más de 3 empleados

```sql
select id_departamento
from empleados
group by id_departamento
having count(*) > 3;
```

✔ Correcto

## 4️⃣ Departamentos con salario medio > 3000

```sql
select id_departamento
from empleados
group by id_departamento
having avg(salario) > 3000;
```

✔ Correcto

# 🔹 BLOQUE 3 – JOIN

## 1️⃣ Mostrar empleado y su departamento

```sql
select e.nombre, d.nombre_departamento
from empleados e
inner join departamentos d
on e.id_departamento = d.id;
```

✔ Corrección aplicada (faltaba alias inicial)

---

## 2️⃣ Departamentos sin empleados

```sql
select d.nombre_departamento
from departamentos d
left join empleados e
on d.id = e.id_departamento
where e.id is null;
```

✔ Uso correcto de LEFT JOIN + IS NULL

---

## 3️⃣ Empleados del departamento "Ventas"

```sql
select e.nombre
from empleados e
inner join departamentos d
on e.id_departamento = d.id
where d.nombre_departamento = 'Ventas';
```

✔ Eliminado GROUP BY innecesario

# 🔹 BLOQUE 5 – EXISTS Y SUBCONSULTAS CORRELACIONADAS

## 1️⃣ Departamentos con empleados

```sql
select d.nombre_departamento
from departamentos d
where exists (
    select *
    from empleados e
    where e.id_departamento = d.id
);
```

✔ Correcto

---

## 2️⃣ Departamentos sin empleados

```sql
where not exists (
    select *
    from empleados e
    where e.id_departamento = d.id
);
```

✔ Correcto

---

## 3️⃣ Empleados que trabajan en "Ventas"

```sql
select e.nombre
from empleados e
where exists (
    select *
    from departamentos d
    where d.id = e.id_departamento
    and d.nombre_departamento = 'Ventas'
);
```

✔ Subconsulta correctamente correlacionada

---

## 4️⃣ Empleados que NO trabajan en "Ventas"

❌ Error inicial: faltaba correlación.

✔ Corrección:

```sql
where not exists (
    select *
    from departamentos d
    where d.id = e.id_departamento
    and d.nombre_departamento = 'Ventas'
);
```

---

## 5️⃣ Empleados en departamentos con más de 2 empleados

### Versión con GROUP BY

```sql
select e.nombre
from empleados e
where exists (
    select *
    from empleados e2
    where e2.id_departamento = e.id_departamento
    group by e2.id_departamento
    having count(*) > 2
);
```

### Versión alternativa sin GROUP BY

```sql
select e.nombre
from empleados e
where (
    select count(*)
    from empleados e2
    where e2.id_departamento = e.id_departamento
) > 2;
```

✔ Se aprendió diferencia de rendimiento entre ambas.
